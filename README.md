Credit Card Transaction And Customer Analysis Report-
This Power BI project analyzes credit card transaction data to track total revenue, interest earned, transaction amount, and transaction volume.
The dashboard provides insights into spending patterns by expenditure type, card category, chip usage, customer demographics, and quarterly trends.
Data was cleaned and modeled using Power Query, with key KPIs and trends calculated using DAX measures.
Interactive slicers and visuals enable quick analysis for financial and business decision-making.

Healthcare Risk Analytics Using Power BI- 
This Power BI project analyzes healthcare data for over 5,000 patients to assess health risks and lifestyle factors.
The dashboard visualizes key indicators such as heart attack and stroke occurrence, smoking habits, obesity, exercise levels, and age-group risk distribution.
Health risk levels are categorized as normal, moderate, high, and critical using DAX measures.
Interactive visuals enable quick identification of high-risk patient groups to support preventive healthcare decisions.

E-Commerce Regional Performance Analyzer-
This project is an interactive Power BI dashboard designed to analyze Amazon sales data across multiple dimensions including sales, profit, orders, and customer segments.
It provides insights into sales by country, category, sub-category, shipping mode, and yearly trends, along with top-performing products.
Advanced DAX measures and Power Query transformations were used to calculate KPIs and ensure clean, accurate reporting.
The dashboard supports data-driven decision-making through dynamic filters, drill-downs, and visual trend analysis.
